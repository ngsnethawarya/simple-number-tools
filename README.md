# Simple Number Tools

A small Python project with two functions:

- `is_even(n)` → checks if a number is even  
- `square(n)` → returns the square of a number  

This is a simple practice project for beginners.

## Usage

```python
from number_tools import is_even, square

print(is_even(10))  # True
print(square(5))    # 25
